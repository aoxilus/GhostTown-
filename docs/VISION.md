# Visión — GhostTown

App **local**: respaldar Gmail en la PC → leer offline → (opcional) limpiar basura comercial con OpenAI.

```
Gmail IMAP → data/ → ghosttown/ → http://127.0.0.1:8765
```

## Fases

| # | Qué | Notas |
|---|-----|--------|
| 1 | Respaldo IMAP + UI tipo Gmail | Obligatoria; sin OpenAI |
| 2 | IA propone borrar comercial | Opcional; **usuario confirma** |

## Regla de oro

Sin respaldo previo, no se borra. Ante la duda → **conservar**. Borrar = mover a Papelera Gmail (reversible).

## Conservar

Fotos/familia, conversaciones humanas, contactos, impuestos, legales, banco, salud, gobierno, escuela/trabajo, adjuntos importantes.

## Candidato a borrar (solo tras confirmar)

Newsletters, promos, ads, marketing automático, no-reply comercial repetido.

## Qué NO es

No es un cliente de correo completo. Solo lectura local + sync + limpieza confirmada.
